Aide
Bienvenue sur le site des chronomètres pour les serveurs possédant un jeu de mot à trouver dans le tchat tous les X temps

Vous devez cliquer sur le bouton start du serveur en question lorsque le message du mot à trouver apparaît pour démarrer le chronomètre qui tournera en boucle tout seul.
Notez qu'un bip sonore se fera entendre 15 secondes avant la fin du chronomètre

Si vous souhaitez que nous ajoutons un serveur possédant la même animation dans son tchat ou nous faire parvenir une suggestion, veuillez nous contacter sur discord via Notrya#7885 ou Hyceman#6969
